How to Use

  1. Put this file in src/main/resources/static/auth-test.html
  2. Start your Spring Boot project
  3. Open http://localhost:8080/auth-test.html
  4. Test your register endpoint and login endpoint
  5. After login, test /auth/me with the saved JWT

If your backend endpoint is not /auth/register or /auth/login, modify the JavaScript URLs below.
